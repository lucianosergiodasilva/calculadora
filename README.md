# 🧮 Calculadora Online - Caso de Estudo

Este projeto é uma calculadora web simples e intuitiva que realiza as operações matemáticas básicas. Ele foi desenvolvido como um caso de estudo para praticar e consolidar conceitos fundamentais de desenvolvimento web front-end.

## 🚀 Como o Projeto Funciona

A calculadora permite realizar as quatro operações básicas da matemática, além de porcentagem e manipulação de decimais:
* **Soma (`+`)**, **Subtração (`-`)**, **Multiplicação (`×`)** e **Divisão (`÷`)**.
* **Porcentagem (`%`)** para cálculos rápidos.
* **Função Limpar (`C`)** para reiniciar o visor.
* **Botão de Apagar (Backspace)** para corrigir o último dígito digitado.
* **Alternador de Temas (Color Themes):** No topo da página, o usuário pode clicar nos círculos coloridos para alterar o esquema de cores e personalizar a interface da calculadora.
